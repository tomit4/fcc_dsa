Problem Statement

Given a row and column wise (nxn) sorted matrix. Write a program to search a key
in a given matrix.

```
    ---------------------
      0    1    2    3
  0 | 10 | 20 | 30 | 40 |
  1 | 15 | 25 | 35 | 45 |
  2 | 27 | 29 | 37 | 48 |
  3 | 32 | 33 | 39 | 51 |
    ---------------------
          matrix[][]
```
